About This Design System

Potting Shed rebrand: A tactile, warm redesign moving from flat web-greens to earthy kraft creams, pine forests, terracotta gifts, and honey highlights. The palette is meant to feel like a botanical gardening space — considered, natural, and inviting.

Token layers:

  • Primitives — raw hues (--pine-700, --clay-600, etc.)
  • Semantic roles — purpose-driven naming (--color-primary, --surface-page, etc.)
  • Legacy aliases — original token names re-mapped to new palette for instant re-skinning

Components: Plain React with no build-tool dependencies. Drop into any project (Vite, CRA, Next.js, or static HTML + CDN).
