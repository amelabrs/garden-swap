# Garden Swap — Bloom Design Handoff

Everything needed to implement the **Bloom** redesign on the Garden Swap website.

## HTML reference pages (open in any browser — no build, works offline)
Cross-linked, so you can click through the whole flow (logo → home, nav → browse, cards → detail, owner → profile).

- **GardenSwap-Bloom.html** — Home (hero, categories, listings, footer)
- **Bloom-Browse.html** — Browse marketplace (filters, sort, card grid, pagination)
- **Bloom-Detail.html** — Plant detail (gallery, tags, owner card, CTAs, more-near-you)
- **Bloom-Profile.html** — User profile (banner, stats, tabs, listings grid)

Every element is inline-styled, so exact colors / spacing / radii / shadows are right there in the markup to copy.

## Docs
- **BLOOM-REFACTOR-PROMPT.md** — Start here. Step-by-step dev prompt for refactoring `web/WebApp.jsx`. Paste into Claude Code.
- **BLOOM-DESIGN-SPEC.md** — Full design spec: palette, type scale, spacing, shadows.
- **BLOOM-HOMEPAGE.md** — Copy-paste JSX for the shared nav, hero, buttons, footer.

## Palette (quick reference)
- Green `#2E7D52` · Cream `#FFFBF5` · Orange `#DD7A2E`
- Honey `#E9B949` / light `#FBE6B0` · Dark text `#1f4a32` · Border `#E7DFC8`
- Type: **Outfit** (headlines, 700–800) + **Figtree** (body)
