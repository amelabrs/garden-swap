Subscription-tier identity pill — the Sprout / Grower / Steward growth metaphor, rendered with line icons (sprout → shrub → trees).

```jsx
<TierBadge tier="sprout" />
<TierBadge tier="grower" />
<TierBadge tier="steward" />
```

Sprout reads mint-green (free), Grower deeper fern, Steward denim (premium). Use on the profile header, paywall comparison, and anywhere elite status appears. `showGlyph={false}` drops the icon for tight inline use.
